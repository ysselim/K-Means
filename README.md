# Parallel K-Means Clustering (OpenMP)

## 1. Project Overview
High-performance implementation of the K-Means clustering algorithm in **C** using **OpenMP** for shared-memory parallelism.  
Includes automated benchmarking and scalability analysis across varying dataset sizes and thread counts on multicore systems.


## 2. Features
- Parallel K-Means using **OpenMP (shared-memory model)**
- Automated benchmarking with **Bash scripting**
- Performance analysis across **1–8 threads** and large datasets (10K–1M points)
- Scalability evaluation using **speedup and runtime metrics**
- Visualization of results using **Gnuplot**


## 3. Tech Stack
- **C**
- **OpenMP**
- **Bash**
- **Gnuplot**
